与Screenfetch相比，neofetch的logo图像非常接近，但是个性化可定制更高，支持信息更多，输出更快。比如logo定制、字体配色等多样玩法

配置

vim ~/.config//neofetch/config.conf
  1. 设置显示logo，具体有哪些logo可以看ls /usr/share/neofetch/ascii/distro
    不知什么原因,我的系统中并没有这个路径,于是我去找到了包含ascii/distro的仓库
git clone https://gitlab.com/ABHIRAMSHIBU/neofetch.git
cd neofetch
sudo cp ascii /usr/share/neofetch/ascii

编辑~/.config/neofetch/config.conf

image_source="path" #将pathi修改为
/usr/share/neofetch/ascii

  1. 将需要显示的信息前的注释符号#去掉
